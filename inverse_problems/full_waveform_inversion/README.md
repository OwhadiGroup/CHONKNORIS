# Full Waveform Inversion (FWI) Experiments 

Authored by Aleksei G. Sorokin

1. `split_train_val.py` is used to split datasets into training and validation sets. 
2. `acoustic_forward_solver.py` contains our forward model, a translation of the [FD Acoustic Modeling Lab by Xin Wang](https://csim.kaust.edu.sa/files/SeismicInversion/Chapter.FD/lab.FD2.8/lab.html). See the header of the file for more info. Run and change the parameters in this file to experiment with the forward solver. 
3. `gauss_newton_solver.py` contains our Gauss-Newton (GN) / Newton--Kantorovich (NK) solver. Run and change the parameters in this file to experiment with the GN/NK solver. 
4. `generate_data.py` uses the GN / NK solver to generate solution trajectories. 
5. `fwi.ipynb` uses the generated data to train CHONKNORIS models and perform inference 
6. `explore.ipynb` performs some analysis / aggregation from data generated by `fwi,ipynb`
